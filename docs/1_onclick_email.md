- Sign in to n8n.
- Create an account for n8n.
- Create a new workflow.
- The workflow starts with the “When chat message received” trigger node.
- When a user sends a message, this trigger activates the workflow.
- The message is then sent to the AI Agent node.
- The AI Agent uses the connected Google Gemini Chat Model to generate a reply.
- The generated response is sent back to the chat window.
- Additional connectors like Memory and Tools can enhance the agent’s capabilities
